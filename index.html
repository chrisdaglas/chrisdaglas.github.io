Περιγραφή προγράμματος

Η HePIS υλοποιεί την πρωτοβουλία «FIT4JOBS» μέσα από την οποία δίνεται η δυνατότητα σε ανέργους να συμμετέχουν σε πρόγραμμα εκπαίδευσης και κατάρτισης χωρίς κόστος σε προχωρημένες δεξιότητες Πληροφορικής. Οι θεματικές ενότητες που καλύπτονται είναι ο προγραμματισμός, η σχεδίαση web και κινητών εφαρμογών, χρησιμοποιώντας λογισμικό και εργαλεία με μεγάλη ζήτηση στην αγορά εργασίας.

Το πρόγραμμα εκπαίδευσης με εξειδίκευση σε .NET & C# περιλαμβάνει τα ακόλουθα αντικείμενα:

FRONT END: HTML5, CSS3, JAVASCRIPT, JQuery, Bootstrap Responsive Web Design, AngularsJS.

BACK END: Application Development in .Net with C# and ASP.MVC in Visual Studio 2015
Database design and Development with SQLSERVER 2014

Στο πλαίσιο της Πανευρωπαϊκής Εβδομάδας Προγραμματισμού «EU Code Week» , η HePIS σε συνεργασία με τη Microsoft Ελλάς διοργανώνουν προγράμματα εκπαίδευσης και κατάρτισης «FIT4JOBS» , διάρκειας 180 ωρών, σε τεχνολογίες Microsoft, όπως η γλώσσα προγραμματισμού C#, το .NET Framework και η ανάπτυξη ASP.NET εφαρμογών.

Παράλληλα, μέσω του προγράμματος, οι υποψήφιοι θα ενισχύσουν τις κοινωνικές τους δεξιότητες όπως η επικοινωνία, η συνεργασία, η ικανότητα διοίκησης και η πρωτοβουλία.

Ο φορέας

H HePIS ιδρύθηκε το 2006 με όραμα να δώσει τη δυνατότητα στους πολίτες της χώρας μας να αξιοποιήσουν στο μέγιστο τις δυνατότητες των Τεχνολογιών Πληροφορικής και Επικοινωνιών (ΤΠΕ) με στόχο την επαγγελματική και κοινωνική τους ανέλιξη.

Αποστολή της HePIS είναι η ανάδειξη του ρόλου των Τεχνολογιών Πληροφορικής και Τηλεπικοινωνιών ως στρατηγικό άξονα για την ανάπτυξη και την ευημερία της χώρας μας, μέσα στο παγκόσμιο ανταγωνιστικό περιβάλλον.
